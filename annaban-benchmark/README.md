# Annaban Benchmark Suite

Multi-model evaluation framework for LLM consensus, routing, and governance scoring.

## Features
- Multi-runner architecture (OpenAI, Anthropic, Gemini, Grok, OpenRouter)
- Consensus + cost metrics
- FastAPI benchmark endpoint
- Kubernetes manifest generator
- Dataset-driven deterministic benchmark tasks

## Install
```bash
pip install -e .
```

## Run
```bash
annaban
```
