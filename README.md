# Annaban Benchmark Suite

Multi-model consensus benchmarking system for GPT, Claude, Gemini, and Grok.

## Key Metrics

- Failure Rate (Byzantine resilience proxy)
- Agreement Stability (semantic consensus)
- Cost per Task
- Governance Score (composite metric)

## Run

```bash
pip install -e .
annaban
```

## Use Cases

- Logistics simulation
- Defense-grade planning systems
- Multi-agent orchestration evaluation
- AI governance testing

## Architecture

- Latency-aware routing
- Semantic consensus engine
- Cost optimization layer
- Kubernetes deployment generator
